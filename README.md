## Поддерживаемые команды

| Команда | Описание | Флаги и специальные символы |
|---------|-----------|--------|
| `ls` | Вывод информации о директории/файле | `-l` (подробно: права доступа, размер, дата изменения, имя файла)|
| `cd` | Изменение текущей директории | `~` (домашний каталог) `..` (родительский каталок)|
| `cat` | Вывод содержимого файла | |
| `cp` | Копирование файлов и директорий | `-r` (рекурсивно) |
| `mv` | Перемещение/переименование файлов и директорий | |
| `rm` | Удаление файлов и директорий | `-r` (рекурсивно) |
| `exit` | Выход из терминала | |

**Каждая команда и результат ее выполнения записываются в shell.log**

## Принятые решения:
* Пустая строка не обрабатывается и не логируется
* Пути с пробелами требуют заключения в кавычки
* Если будет одна не закрытая кавычка, выведется ошибка. Однака если указать кавычку внутри других кавычек ("'"), она будет считаться частью имени директории/файла
* Символ `\` не является экранирующим, а считается частью пути
* Поддержка путей в форматах Windows и Linux
* Флаги (-l, -r) можно указывать в любом месте команды

**ls**
* Поддержка вывода информации для нескольких файлов/директорий
* Подробный формат (`-l`) включает: права доступа, размер, дату изменения, имя
* Автоматическая сортировка содержимого директорий

**cd**
* Требует ровно один аргумент - путь
* Поддержка специальных символов `~` (домашняя директория) и `..` (родительская)

**cat**
* Требует ровно один аргумент - путь к файлу
* Используется кодировка UTF-8 для поддержки русского языка

**cp**
* Требует два аргумента: источник и цель
* Запрещено копирование директории в саму себя
* Рекурсивное копирование требует флага `-r`

**mv**
* Требует два аргумента: источник и цель
* Запрещено перемещение директории в саму себя

**rm**
* Требует один аргумент - путь
* Запрещено удаление текущей или родительской директории
* Запрещено удаление корневого каталога
* Удаление директорий требует подтверждения пользователя
* Статус `CANCELLED` при отмене удаления
